# Dashboard de Vendas: Xbox Game Pass Subscriptions
Este projeto consiste em um dashboard interativo no Excel desenvolvido para analisar o desempenho de vendas e assinaturas do Xbox Game Pass. O objetivo é transformar dados brutos em insights visuais claros para apoiar a tomada de decisão.

# Como Visualizar

1 - Baixe o arquivo .xlsx deste repositório.
2 - Abra no Microsoft Excel.
3 - Utilize os filtros para explorar diferentes cenários de vendas.
